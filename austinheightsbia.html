They wanted a WordPress website that allowed them to not only showcase what they do as an association, but also a business directory that includes all the businesses in the area. This website was made using a child theme that they purchased.

The Process

During the inintal client meetings, they specified that they wanted a clean, easy-to-navigate, and functional website, as their old website stopped working. The main purpose of this website is to have a business directory so I found a plugin called "GeoDirectory" and customized it in accordance to Austin Heights BIA's needs. This directory includes over 250 businesses with a unique profile for each.

Based on the budget and time constraints, this was built using the WPBakery Page Builder. This showed me a completely new way of designing websites. Instead of configuring and creating new PHP files, this allowed users to build pages much more easily. Although it is not as customizable, it is definetly much faster and cheaper. This is the first time I've used a page builder and it has certainly opened my eyes in terms of having different ways to building a website. Not all clients have the time and budget to have thousands of code that are written exclusively for them. Sometimes using a pre-existing theme or plugin can be the perfect solution for a client.
