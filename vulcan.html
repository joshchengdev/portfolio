Vulcan

Vulcan was the first web group project that I’ve ever done, which required the use of Git. The purpose of this site was to practice my HTML and CSS skills. I was given a mockup of what the website should look like and was asked to build it from scratch. This was also the first time I’ve used SASS on a big project.

The Process

As I was styling through the website, I realized how useful and convenient SASS was. Using variables for specific sizes, colors, and media queries were extremely helpful and time efficient since a lot of the styles on the website followed a similar pattern.

                
$head-font: 'Roboto Condensed', sans-serif;
$body-font: 'Roboto', sans-serif;
$button-color: #429bd6;
$dark-blue: #154a66;
$green: #367d22;
$gray: #616161;
$selection: #bbbbbb;
$news: #bedcef;
$dark-gray:#504f4f;
$light-gray: #c5c5c5;
$light-blue: #009ddc;

// Media Query Breakpoints
$desktop-wide: 1200px;
$desktop-narrow: 1140px;
$tablet-landscape: 1024px;
$tablet-portrait: 768px;
$phone-landscape: 560px;
$phone-portrait: 480px;

What Can Be Improved:

  • Use variables in the media query
  • Use mixins to create quick styles
  • Use placeholder selectors for outputting DRY CSS
  • Improve form style at mid-screen sizes
  • Start mobile first rather than desktop
